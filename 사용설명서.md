# 내비온 견적서 생성기 — 사용 설명서

정부 학술용역 단가 기준으로 견적을 계산하고, 회사 HWP 양식 그대로 **HWP·PDF를 자동 생성**하는 데스크톱 프로그램입니다.

## 주요 기능
- **엑셀 계산 모델 1:1 재현** — 인건비(2026 학술용역단가)·경비·일반관리비(5%)·이윤(10%)·부가세(10%)·절삭
- **이윤 포함 / 이윤 없는 버전** 토글
- **목표금액 → 인건비 참여율 자동 역산** (만원미만 자동 절삭으로 목표금액 정확히 일치, 직급별 🔒값 고정 가능)
- **기존 견적서 폴더 관리** — 폴더 선택 시 기존 .hwp를 스캔해 용역명·금액·일자 카드로 표시, 열기/PDF
- **AI 자동 초안** (Google AI Studio Gemini, 무료 키) — 용역 설명만 입력하면 인력·경비 구성 제안 (금액은 검증된 엔진이 확정)
- **HWP·PDF 출력** — 직인·로고·공급자 정보가 들어간 회사 양식 그대로

---

## 설치 (최초 1회)

1. **한글(HWP) 설치 확인** — 한글 2022/2024가 설치되어 있어야 합니다 (PDF 변환 포함).
2. 명령 프롬프트에서 의존성 설치:
   ```
   pip install -r requirements.txt
   ```

## 실행

`견적서생성기.bat` 더블클릭 (또는 `python app.py`).

> 견적서를 **생성할 때 한글이 켜져 있으면 닫아 주세요** — COM 자동화와 충돌할 수 있습니다.

## 사용 흐름

### 1. 폴더 선택
대시보드 상단 **폴더 선택**으로 견적서를 저장·관리할 폴더를 지정합니다. 폴더 안의 기존 `.hwp` 견적서가 자동으로 카드로 나타납니다.

### 2. 새 견적서
- **＋ 새 견적서** → 직접 작성
- **▾ → ✦ AI 초안으로 시작** → 과업지시서(RFP)/용역 설명으로 AI 구성 제안

### 3. 입력
- **문서 정보**: 수신처·견적번호(자동 발급 가능)·일자·용역명·기간
- **견적 설정**: 이윤 포함/없음, 목표금액, 역산 모드
- **인건비**: 직급별 명수·참여율·기간 (단가는 단가표에서 자동)
- **경비**: 엑셀형 표(구분·내역·수량·단가·금액). **수량은 "5명"/"1식"처럼 표기**하면 앞 숫자가 금액 계산에 자동 반영(예: 단가×5)되고, "-"나 숫자 없는 표기는 단가만 적용됩니다. 내역은 한 줄에 하나씩 "- "로 시작.

오른쪽 **미리보기**가 실시간으로 갱신됩니다. 목표금액을 넣으면 **예산 가이드**가 적정 인건비·경비 범위를 보여줍니다.

### 4. 목표금액 맞추기
목표금액 입력 → **⚖ 참여율 자동 역산** 클릭 → 인건비 참여율이 자동 계산되고 끝수는 절삭으로 보정되어 견적금액이 목표와 정확히 일치합니다.
- *균등 참여율*: 모든 직급 동일 참여율
- *현재 비율 유지*: 직급별 참여율 비율을 유지한 채 스케일

### 5. 생성
- **HWP 생성** — .hwp만
- **HWP + PDF 생성** — 둘 다 (생성 후 PDF 자동 열림)
- **임시 저장** — `.quote.json`으로 저장 (나중에 카드에서 **재편집** 가능)

파일명: `견적서_{용역명}_{YYMMDD}.hwp / .pdf / .quote.json`

### 6. 견적서 삭제
대시보드 카드 우상단 **×** → 삭제 확인 창.
- 기본: **목록(재편집 데이터 .quote.json)에서만 제거** — 폴더의 실제 한글/PDF 파일은 **그대로 남습니다**.
- **"폴더의 실제 파일도 함께 삭제"** 체크 시에만 .hwp/.pdf까지 삭제합니다.

## AI 자동 견적 (과업지시서 기반)
**✦ AI 초안** 또는 새 견적의 **AI 초안으로 시작**을 누르고, **과업지시서(RFP) 전문을 붙여넣거나** 용역을 설명한 뒤 목표금액을 입력하면, AI가 그 내용을 분석해 인력 구성·경비 항목·내역 문구를 제안합니다. **금액 계산은 AI가 하지 않고** 검증된 엔진이 목표금액에 맞춰 확정하므로 숫자는 항상 정확합니다. 불러온 초안은 자유롭게 수정 후 생성하세요.

## Google Drive 연동 (선택)
생성한 견적서를 드라이브에 자동/수동 업로드합니다.
1. [Google Cloud Console → 사용자 인증 정보](https://console.cloud.google.com/apis/credentials)에서 **OAuth 클라이언트 ID(데스크톱 앱)** 생성 → `client_secret.json` 다운로드
2. 그 파일을 **프로그램 폴더**에 `client_secret.json` 이름으로 저장
3. **설정 → Google Drive 연동 → Drive 연결** → 브라우저에서 구글 로그인·허용
4. **업로드 폴더명** 지정, 필요시 **견적 생성 시 자동 업로드** 체크
- 권한 범위는 `drive.file`(이 앱이 만든 파일만)이라 기존 드라이브 파일에는 접근하지 않습니다.
- 라이브러리 미설치 시 Drive 기능만 비활성화되고 나머지는 정상 동작합니다(`pip install google-api-python-client google-auth-oauthlib`).

## AI 설정 (선택)
1. https://aistudio.google.com 에서 무료 API 키 발급
2. **설정(⚙) → AI** 에 키 입력 → **모델 선택** → **키·모델 저장** → **키 확인**
3. **모델 드롭다운**: 권장값은 `Gemini Flash (항상 최신)`이며, `Gemini 3.5 Flash`, `Gemini 3 Flash`, **`Gemini 3.1 Flash-Lite (최저가)`**, `Gemini 2.5 Flash` 중에서 고를 수 있습니다. **목록 새로고침**을 누르면 내 키로 실제 사용 가능한 모델이 추가되고, `(직접 입력…)`으로 임의 모델명도 지정 가능합니다.
4. 키는 Windows 계정에 암호화(DPAPI)되어 저장됩니다. 키가 없으면 AI만 비활성화되고 수동 입력은 정상 동작합니다.

> 참고: `gemini-2.0-flash`는 2026-06-01 종료됐습니다. 구형 모델이 종료돼도 `Gemini Flash (항상 최신)`을 쓰면 자동으로 현재 모델로 연결됩니다.

## 단가표 수정
**설정 → 학술용역 인건비 단가표**에서 연도별 직급 단가를 수정/추가할 수 있습니다. 새 연도 단가가 고시되면 **＋ 연도 추가** 후 값을 입력하세요.

## 문제 해결
- **설정 → 진단 → 환경 점검**: 한글 COM·보안모듈·템플릿·키·폴더 상태 확인
- **한글 구동 테스트**: 실제 한글 실행이 되는지 점검 (느림)
- 생성이 멈추면(60초 초과) 자동 중단되며, 한글 창이 떠 있으면 닫고 재시도하세요.

## 구성 (개발자용)
```
app.py                  진입점 (pywebview)
src/engine/             계산(calc) · 역산(goalseek) · 한글금액(money_kor)
src/hwp/                템플릿화(make_template) · 생성(hwp_writer) · 필드매핑(field_map)
src/scan/hwp_scan.py    기존 HWP 메타 스캔 (olefile)
src/ai/gemini.py        Gemini REST
src/store/              설정·키 암호화 · 견적 JSON
ui/                     index.html · app.css · app.js
templates/견적서_템플릿.hwp   필드 삽입된 양식 (원본 보존)
tests/                  pytest (계산/역산/스캔/필드/라운드트립)
```
검증: `pytest`(엔진 51건), `pytest -m hwp`(실제 한글 생성 라운드트립).
